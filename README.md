# Sentiment Analysis of Philippine Election Tweets

## Model Weights Download

Due to file size restrictions, the trained deep learning models (XLM-RoBERTa, mBERT, and EmoBERT) are not included directly in this repository. 

To run the classification pipeline and the Graphical User Interface (GUI), please download the required model weights from the following link:

**🔗 [https://drive.google.com/drive/folders/14FpYsQ_xFivPCpoz2rkRBoHvDBZGoccx?usp=sharing]**

### Setup Instructions

1. Download the exported model files from the provided link.
2. Place the downloaded `.pt` and `.pth` weight files into the `models/` directory (or directly into the root folder where the script is located).
3. Ensure you have the necessary dependencies installed (`torch`, `transformers`, etc.).
4. Run the application script to launch the GUI.
